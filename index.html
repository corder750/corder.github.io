SeatMaster

A simple, visual restaurant seat management tool built in Code.org App Lab. Track booths, indoor tables, and patio seating in real time.

Open Live App See Features

Runs on Code.org App Lab – no install required.

At a glance

  • Tap tables to toggle open / occupied in real time
  • Separate layout for booths, indoor, and patio
  • List view with filters and seat counts

Features

Interactive Floor Map

Each button on the map represents a real table. Tap to switch between open (green) and occupied (red) with semi-transparent colors.

List Mode & Filters

Switch to List Mode to see every table with its type, seats, and status. Use the filter menu to show only open, occupied, booths, indoor, or patio tables.

Live Counters

Automatic counters show how many tables are open versus occupied so staff can see capacity at a glance.

Simple Login Gate

Basic login screen to keep random users out of the manager view during demos or class presentations.

How It Works

  1. Log in using the demo username and password.
  2. Tap tables on the floor plan to mark them open or occupied.
  3. Switch to List Mode to see a scrollable list of all tables with seats and status.
  4. Use the filter menu to view only open tables, only occupied tables, or by area (booths, indoor, patio).
  5. Watch the counters at the top update as you change table status.

SeatMaster is a prototype built for learning and demo purposes using Code.org App Lab. It’s great for school projects, early product concepts, and experimenting with restaurant-style seat management.
