Update Showcase

A timeline of SeatMaster changes, new features, and visual improvements.

v1.0 – Initial Prototype

First working version built in Code.org App Lab:

  • Basic floor map buttons for tables
  • Open / occupied status toggles
  • Simple counters for open vs occupied tables

v1.1 – List Mode & Filters

Added a table list view and filter options:

  • List Mode with all tables + seat counts
  • Filters for open, occupied, booths, indoor, patio
  • Better color coding for table types

v1.2 – Demo Login Screen

Created a simple login gate for demos:

  • Username: Admin
  • Password: SeatMaster2026
  • Prevents random changing tables during class demos
